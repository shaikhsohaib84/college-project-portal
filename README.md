# College Project Portal

## 📌 Description
A Django-based project management system where students can submit projects, teachers can review them, and admins can manage the platform.

---

## 🚀 Features
- User Authentication (Student, Teacher, Admin)
- Project submission system
- Role-based dashboards
- Project review system

---

## 🛠️ Tech Stack
- Python
- Django(ORM)
- SQLite
- HTML/CSS

---

## ⚙️ How to Run

```bash
git clone https://github.com/shaikhsohaib84/college-project-portal.git
cd college-project-portal

python3 -m venv .venv
source .venv/bin/activate

pip install -r requirements.txt

python manage.py migrate
python manage.py runserver
